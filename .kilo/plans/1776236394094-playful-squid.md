# Plan: Explore as Guest Feature

## Overview
Add ability for users to explore the app without authentication. Guests can view the homepage with channels and livestreams but cannot access profile, chat, or other authenticated features.

## API Endpoints (No Auth Required)
- `GET /companies?limit={limit}&offset=0` → returns all channels
- `GET /livestreams/active?limit={limit}&offset=0` → returns all active livestreams

## Guest User Flow
1. User taps "Explore as Guest" on login screen
2. Navigated to homepage in guest mode
3. Sees horizontal channel slider + horizontal livestream slider
4. Can tap to view company details (read-only)
5. Trying to access profile/chat → shows dialog with:
   - "Sign in to access this feature"
   - "Sign In" button → goes to login screen
   - "Continue as Guest" button → dismisses and returns to browsing
